# Информационная система медицинской клиники

## Описание
Приложение для автоматизации работы регистратуры. Позволяет вести учет пациентов, врачей и запись на прием.

## Функционал
- Авторизация и регистрация с CAPTCHA и временной блокировкой
- 3 роли пользователей: admin, operator, user
- CRUD операции для пациентов, врачей и приемов
- Автоматический расчет стоимости (скидка 20% на повторный прием)
- Журнал действий пользователей

## Технологии
- C# / .NET 8.0
- Windows Forms
- SQLite (Microsoft.Data.Sqlite)

## Инструкция по запуску
1. Установите Visual Studio 2022+
2. Клонируйте репозиторий
3. Восстановите пакеты NuGet (Microsoft.Data.Sqlite)
4. Запустите проект (F5)
